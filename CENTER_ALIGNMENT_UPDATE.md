## 🎨 使用的 CSS 類別

### 文字居中
```css
text-center          /* 文字居中 */
max-w-3xl mx-auto   /* 最大寬度並水平居中 */
```

### Flexbox 居中
```css
flex flex-col items-center  /* 垂直 Flex，項目居中 */
justify-center              /* 水平居中 */
```

### Grid 居中
```css
justify-items-center  /* Grid 項目在列中居中 */
```

### 列表居中
```css
text-left w-full           /* 列表文字左對齊，但... */
justify-center             /* ...容器居中 */
```
